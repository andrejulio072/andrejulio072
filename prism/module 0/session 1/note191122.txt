Linux commands
1. ls
The ls command is used to list files and directories in the current working directory

2. pwd
The pwd command allows you to print the current working directory on your terminal

3. cd
Change directory
	cd <name_of_directory>	-> to switch to particular directory
	cd ..					-> go one directory up

4. mkdir
The mkdir command allows you to create directories from within the terminal
	mkdir <name_of_directory>

5. touch
touch command is used to create a file
	touch <name_of_file>.<extension>

6. cp copy files and folders
	cp <source> <destination>

7. mv move files and folders
	mv <source> <destination>

8. rm
	rm <name_of_file> 			-> deletes files
	rm -r <name_of_directory>	-> deletes directory
